
## Agent skills

### Issue tracker

Issues live in GitHub Issues; external PRs are treated as a request surface and run through the same triage flow. See `docs/agents/issue-tracker.md`.

### Triage labels

All five canonical roles use their default label strings: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout: one `CONTEXT.md` at root + `docs/adr/`. See `docs/agents/domain.md`.
